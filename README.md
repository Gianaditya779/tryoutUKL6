Program yang saya buat ini digunakan untuk melakukan penjumlahan dua buah matriks berukuran sama berdasarkan input dari user.
1.Program meminta user untuk menginput jumlah baris dan kolom matriks;
2.program akan membuat tiga buah array dua dimensi : matriksA, MatriksB, Hasil yang ukurannya sesuai dengan yang di input oleh user;
3.program akan meminta user untuk menginput nilai elemen satu persatu matriks;
4.setelah kedua matriks terisi, prgram akan menjalankan penjumlahan;
5.Selanjutnya program akan menampilkan output berupa hasil dari penjumlahan matriks tersebut.
